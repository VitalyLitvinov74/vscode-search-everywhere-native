# Поиск в стиле Search Everywhere для VS Code

> Один привычный вход к родным режимам поиска VS Code: файлы, символы, текст и действия.

Проект исследует пользовательский сценарий уровня `Search Everywhere` из PhpStorm, но не создаёт собственный поисковый движок. Расширение должно использовать результаты, ранжирование, исключения и поддержку удалённых сред, которые уже предоставляет VS Code.

Статус: сбор информации и проектирование. Реализация ещё не начата.

## Главное решение

Расширение не будет:

- строить или хранить собственный индекс;
- сканировать рабочую область отдельным процессом;
- запускать собственный `ripgrep`;
- читать и разбирать файлы ради дублирования встроенного поиска;
- зависеть от предлагаемого API, запрещённого для публикации в Marketplace.

Вычисление результатов остаётся у VS Code. Ценность расширения — единый, предсказуемый и клавиатурный пользовательский путь поверх:

- `Quick Open` для файлов и, при включённой настройке, символов;
- `Quick Search` для текста рабочей области;
- поиска символов рабочей области;
- палитры команд;
- полной панели `Search` для регулярных выражений, масок, исключений и областей.

## Целевой пользовательский опыт

- двойной Shift или настраиваемое сочетание открывает поиск;
- выбранный текст или слово под курсором можно передать в запрос;
- пользователь выбирает категорию, не запоминая префиксы `#`, `%` и `>`;
- запрос передаётся в соответствующий родной режим VS Code;
- просмотр, подсветка, открытие рядом и переходы выполняет VS Code;
- сложный текстовый запрос одним действием переносится в полную панель `Search`;
- расширение не изменяет пользовательские сочетания и настройки без явного выбора;
- локальные проекты, WSL, SSH и контейнеры работают через штатную архитектуру VS Code.

## Граница первой версии

Первая версия — тонкий диспетчер родных режимов поиска с проверкой совместимости. Она не обещает единый смешанный список файлов, текста, символов и действий: стабильный API VS Code пока не позволяет расширению получать текстовые совпадения для такого списка.
